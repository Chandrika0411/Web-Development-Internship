# Responsive Landing Page

A modern landing page developed using HTML, CSS, and JavaScript that adapts smoothly to different screen sizes. The project focuses on creating an engaging user experience with an interactive navigation bar and responsive design principles.

## Key Features

- Responsive layout for desktop, tablet, and mobile devices
- Fixed navigation bar with scroll effect
- Hover animations for navigation links
- Smooth scrolling between sections
- Clean and modern user interface

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Getting Started

1. Clone or download this repository.
2. Open the project folder.
3. Run 'index.html' in your preferred web browser.

## Skills Gained

- Responsive Web Design
- CSS Flexbox and Media Queries
- JavaScript DOM Manipulation
- Scroll events
- Interactive UI Design
