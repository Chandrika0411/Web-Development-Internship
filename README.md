# Web Development Internship

This repository contains the projects completed as part of my **Web Development Internship at SkillCraft Technology**. Throughout this internship, I worked on various frontend development projects that helped strengthen my understanding of HTML, CSS, and JavaScript while building responsive and interactive web applications.

Each project focuses on improving practical web development skills, including responsive layouts, DOM manipulation, event handling, user interface design, and application logic.

## Projects Included

### Task 01 – Responsive Landing Page
Designed a modern and fully responsive landing page with a fixed navigation bar, smooth scrolling, hover effects, and an attractive user interface.

### Task 02 – Calculator Web Application
Developed a responsive calculator capable of performing basic arithmetic operations with keyboard support, input validation, and error handling.

### Task 03 – Quiz Game Application
Created an interactive quiz application featuring multiple-choice questions, randomized options, a countdown timer, automatic score calculation, and a final performance summary.

### Task 04 – To-Do Web Application
Built a task management application that allows users to add, edit, complete, and delete tasks through a clean and responsive interface.
