# Calculator Web Application

A responsive calculator built with HTML, CSS, and JavaScript that performs standard arithmetic operations. The application provides a clean interface with keyboard support, error handling, and smooth button interactions.

## Key Features

- Basic arithmetic operations
- Keyboard input support
- Backspace and Clear functions
- Error handling for invalid expressions
- Responsive design for different devices
- Interactive button animations

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Getting Started

1. Download or clone this repository.
2. Open "index.html" in your browser.
3. Use the calculator for performing calculations.
